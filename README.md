# reactnative-bottomsheet
An iOS-esque bottom sheet navigation component

Use like:
```
<BottomSheet style={styles.bottomSheet} onPress={() => props.navigation.pop(1)}>
  <Text>Hello world</Text>
</BottomSheet>
```
